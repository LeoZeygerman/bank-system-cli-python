Bank System CLI (Python)

Простой банковский терминал, написанный на Python.

Проект был создан в учебных целях для практики backend-логики, ООП и работы с JSON.

Возможности проекта

* Регистрация пользователя
* Вход в аккаунт
* Просмотр баланса
* Пополнение баланса
* Снятие денег
* Перевод средств
* История переводов
* Сохранение данных в JSON-файлы

Что использовалось

* Python
* ООП (классы и методы)
* JSON
* CLI (работа через терминал)

Что я практиковал в проекте

Делал

* Разделение проекта на несколько файлов
* Работу с классами и методами
* Создание терминального меню
* Логику авторизации
* Сохранение и загрузку данных
* Обновление данных пользователя
* Обработку ошибок через try/except
* Работу со словарями и списками
* Простую backend-логику

Не делал

* Базы данных
* FastAPI
* Docker
* Настоящую систему безопасности
* Хеширование паролей

Структура проекта

* main.py — запуск программы и авторизация
* bank_menu.py — банковское меню
* models.py — класс Bank и методы
* storage.py — работа с JSON-файлами
